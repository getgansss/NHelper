# NHelperV4

**NHelperV4** — многофункциональная утилита для восстановления и управления системой Windows, особенно эффективная в среде восстановления WinRE.

> ⚠️ **Важно:** Скачивая программу, вы соглашаетесь с [пользовательским соглашением](https://nhelper.gt.tc). Используйте на свой страх и риск.

## 📋 Возможности

### 🚀 Автозагрузка
Управление всеми типами автозагрузки Windows:
- **Реестр** — Run, RunOnce, Winlogon, AppInit_DLLs, CmdLine
- **Папка автозагрузки** — стандартная папка Windows
- **Планировщик задач** — управление задачами
- **Службы** — запуск, остановка, удаление служб

### 🔓 Снятие ограничений
Автоматическое сканирование и снятие блокировок:
- ScancodeMap (переназначение клавиш)
- Debuggers (подмена приложений)
- DisallowRun (запрет запуска программ)
- HostsFile (блокировка сайтов)

### 🛠️ Дополнительные возможности

#### Команды системы
| Функция | Описание |
|---------|----------|
| Перезагрузить ПК | Перезапуск системы |
| Выйти из пользователя | Завершение сеанса |
| Войти в WinRE | Загрузка среды восстановления |
| Выполнить | Диалог запуска программ (Win+R) |
| Вернуть русский язык | Восстановление раскладки |
| Вернуть стандартный шрифт | Сброс системного шрифта |
| Сохранение WinRE | Создание/восстановление среды восстановления |
| Включить UAC | Активация контроля учётных записей |
| sfc /scannow | Проверка целостности системных файлов |
| Полный доступ к файлу | Разблокировка системных файлов |
| Восстановить LogonUI | Восстановление экрана входа |
| Выключить тестовый режим | Отключение загрузки драйверов без подписи |
| Заменить sethc/utilman | Замена утилит специальных возможностей |
| Экстренное восстановление | Расширенное восстановление системы |
| Удобный запуск | Добавление в PATH и контекстное меню |
| Alt+N | Запуск утилиты по нажатию горячей клавиши |
| Очистка драйверов | Удаление сторонних драйверов |

#### Встроенные утилиты
- **Редактор реестра** — альтернатива regedit
- **Проводник** — альтернатива explorer.exe
- **Диспетчер задач** — расширенный taskmgr (заморозка процессов, снятие флага "критический")
- **mbrRE** — резервное копирование и восстановление MBR
- **Браузер** — базовый веб-обозреватель
- **Управление пользователями** — создание, удаление, сброс пароля
- **Очистка системы** — temp, корзина, кэш браузеров, автозагрузка
- **Управление ассоциациями** — восстановление ассоциаций файлов (.exe, .bat, .txt, .lnk, .html и др.)
- **Сброс пароля** — смена пароля пользователя на "1"
- **Удаление дисков** — управление дисками системы

### Запуск в WinRE (среда восстановления)
1. Загрузитесь с установочной флешки Windows
2. Откройте командною строку
3. Впишите в нее regedit, нажмите в верхнем левом углу загрузить куст
4. В данном окне найдите утилиту, после нажмие ПКМ и запустить

### Удобный запуск (после активации)
```cmd
# В командной строке
nh

# Или через контекстное меню (ПКМ по файлу/папке)
```
